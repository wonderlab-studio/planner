# Полезные ссылки

Чтобы меньше платить за сервисы

**free-for-dev**
Коллекция бесплатных инструментов для разработки: хостинг, базы данных, почта, мониторинг и многое другое.

**public-apis**
Большая база бесплатных API, если нужно добавить в свой проект реальные данные: от погоды до финансов.

**awesome-selfhosted**
Подборка open source альтернатив платным сервисам. Можно найти аналоги привычных инструментов и запускать их самостоятельно.

## 🤖 Для ИИ-агентов и вайбкодинга

**awesome-claude-code**
Готовые скиллы, плагины и расширения для Claude Code.

**skills-anthropic**
Официальная коллекция скиллов от Anthropic — хорошая точка старта для работы с агентами.

**awesome-mcp-servers**
Каталог MCP-серверов, которые помогают подключать ИИ-агентов к внешним инструментам.

**awesome-llm-apps**
Готовые примеры AI-приложений и агентов с исходным кодом.

## 🧠 Чтобы лучше понимать ИИ-инструменты

**system-prompts-and-models-of-ai-tools**
Подборка системных промптов популярных AI-сервисов. Можно изучать, как устроены разные продукты.
